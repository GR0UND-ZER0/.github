# Support

## Where to ask for help
- For usage questions and guidance, open a GitHub Issue in the relevant repository.
- For feature ideas, use the feature request template.
- For reproducible problems, use the bug report template.

## Community support
Join the [Ground Zero community chat](https://m.me/j/AbbxfofRAX3TyND-/?send_source=gc%3Acopy_invite_link_c) for discussion and collaboration.

## Before opening an issue
- Search existing issues first
- Provide clear steps, expected behavior, and actual behavior
- Include environment details when relevant

## Security issues
Do not file security vulnerabilities publicly.
Follow the process in SECURITY.md for responsible disclosure.
