# Contributing to Ground Zero

Thanks for contributing.

## Community rules
- Respect everyone in the community
- Keep everything tech-related
- No spam, ads, or scams
- Share useful code, tools, or ideas
- Help others when you can
- Stay consistent and active

## Contribution workflow
1. Open an issue with context, expected outcome, and scope.
2. Create a branch with a clear name.
3. Keep pull requests focused and small.
4. Add tests or validation notes when applicable.
5. Request review and respond to feedback quickly.

## Quality standards
- Prefer readable, maintainable code over clever code.
- Keep docs updated for any behavior changes.
- Avoid committing secrets, keys, or credentials.
