# Ground Zero Organization Defaults

This repository defines organization-wide defaults and shared community standards.

## Repository layout
- profile/README.md: Organization public profile content
- .github/ISSUE_TEMPLATE/: Issue templates and issue template config
- pull_request_template.md: Default PR template for repositories
- CONTRIBUTING.md: Contribution workflow and expectations
- CODE_OF_CONDUCT.md: Community behavior standards
- SECURITY.md: Vulnerability reporting policy
- SUPPORT.md: Support and help channels

## Notes
The inner .github folder is expected: GitHub uses it for issue template configuration even though this repository itself is also named .github.
