# Ground Zero Organization Defaults

This repository defines organization-wide defaults and shared community standards.

## Repository layout
- [profile/README.md](profile/README.md): Organization public profile content
- [.github/ISSUE_TEMPLATE](.github/ISSUE_TEMPLATE): Issue templates and issue template config
- [pull_request_template.md](pull_request_template.md): Default PR template for repositories
- [CONTRIBUTING.md](CONTRIBUTING.md): Contribution workflow and expectations
- [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md): Community behavior standards
- [SECURITY.md](SECURITY.md): Vulnerability reporting policy
- [SUPPORT.md](SUPPORT.md): Support and help channels
